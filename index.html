1. Project Overview — The Vision

Mind Tune brings astrology closer to people through intelligent automation — turning personal birth data into personalized daily insights. It merges data automation, LLM‑based text generation, and a seamless user experience on LINE Official Account.

Objective

  • Automate astrology forecast workflows end‑to‑end
  • Deliver personalized experiences via LINE OA
  • Integrate AI/LLM to enhance content and productivity

Role

  • System Designer & Automation Developer
  • LLM Integration Engineer

Tech Stack

n8n LINE Messaging API Google Sheets (DB) Python Regex OpenAI API

2. Problem & Solution

Before

  • Manual data entry (birth date, time, location)
  • Human verification & daily content delivery
  • Time‑consuming and error‑prone

After — with Mind Tune

  • Automated data normalization (Regex + Python)
  • Token‑based subscription lifecycle (purchase → verify → deliver)
  • n8n orchestrates input → AI generation → LINE delivery

Impact: Reduced manual operations by 100%; consistent daily delivery; higher personalization & engagement.
